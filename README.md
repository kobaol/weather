# Prognoza pogody – React
Aplikacja webowa napisana w React, służąca do wyświetlania prognozy pogody dla wybranych miast.  
Projekt wykonany w ramach kursu **Programowanie Frontend**.

## Funkcjonalności

 Wyświetlanie listy miast (min. 5)
 Szczegółowy widok prognozy pogody dla wybranego miasta
 Aktualna temperatura i warunki pogodowe (ikony)
 Prognoza pogody na 5 dni
 Informacje o opadach (prawdopodobieństwo, rodzaj, ilość)
 Prędkość i kierunek wiatru
 Stopień zachmurzenia
 Wyszukiwanie miast po nazwie
 Oznaczanie miast jako ulubione ⭐
 Osobna podstrona z ulubionymi miastami
 Globalna zmiana jednostek temperatury (°C / °F / K)
 Zachowanie ustawień w localStorage

## Technologie

 React
 React Router
 Redux Toolkit
 JavaScript (ES6+)
 CSS
 localStorage
